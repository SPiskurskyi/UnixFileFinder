#UnixFileFinder

UnixFileFinder provides fast file searching on Unix systems using multiple threads.

## Getting Started

### Prerequisites

- g++ complier
- GNU Make

### Installation & Building

Installation:

```sh
https://github.com/Zeerhiy/UnixFileFinder.git
cd UnixFileFinder
```
For build app:

```sh
make
```

For build only library:

```sh
make libfilefinder.a
```

## Usage

```sh
cd /build
./app <search file>
```
or

```sh
cd /build
./app
<search file>
```
